## 关于本站

本站使用的是`vuepress`构建的，感兴趣的小伙伴可以私聊折腾一下。</br>
用来做个闲置的个人网站倒是合适。没有配置花里胡哨的主题，原因是懒。</br>
本站创建的目的是为了减轻本人毕设接单的一些常见工作及问题。</br>
比如：环境部署、小白入门、个人简介...如果有什么好的意见，欢迎留言！</br>
