# 关于本站

本站使用的是vuepress构建的，感兴趣的小伙伴可以私聊折腾一下。
用来做个闲置的个人网站倒是合适。没有配置花里胡哨的主题，原因是懒。
本站创建的目的是为了减轻本人毕设接单的一些常见工作及问题。
比如：环境部署、小白入门、个人简介...如果有什么好的意见，欢迎留言！

# 问题

毕设费时费力，如果换算成时薪那真是亏麻了。那为什么要接单？
我的回答是：找点事情干、为以后做知识付费铺垫经验，开展第二副业。
诚然，我的工资可以让我很好的在南昌这个二线城市体面的活着。
但是总觉得下班后要做些什么的想法促使着我做了这一切。
我喜欢讲课，其中的成就感是不言而喻的。
一切都随心，随缘。想到什么去做什么就好了。这也是创建本站的缘由。
